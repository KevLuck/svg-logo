# svg-logo

Generates a basic logo based on pre-set options

## Description

A command-line logo generator that allows you to input the logo specifications you want and generate your logo. The logo specifications include the text, color, and shape of the logo. The logo is generated as an SVG file.

### List of Features 

1. Choose 3 letters: The user can choose 3 letters to be included in the logo.
2. Choose a logo shape: The user can choose a shape for the logo, such as a circle, triangle, or square.
3. Choose colors: The user can choose colors for the logo. 
4. Jest function testing package: The application includes a Jest function testing package to help ensure the application is working correctly.


### Usage

To use the application, the user can follow these steps:

1. Choose 3 letters to be included in the logo.
2. Choose a shape for the logo.
3. Choose colors for the logo.
4. Click the "Generate Logo" button.
5. The application will then generate a logo with the chosen letters, shape, and colors.
